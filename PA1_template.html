What is the average daily activity pattern?

First, we calculate the average number of steps taken in each five-minute interval across all days. Then, we make a time series plot (type=“l”) of average number of steps taken in these intervals.

act$interval_steps <- tapply(act$steps, act$interval, mean, na.rm=T)

Imputing missing values

There are a number of days/intervals where there are missing values (NAs). This may introduce bias into calculations or data summaries. To address this, we first calculate and report the total number of missing values in the dataset.

sum(is.na(act$steps))

## [1] 2304

Second, we fill in all of the missing values in the dataset in a new variable, steps2. We replace the missing values in a particular interval with the rounded mean of the steps in the same five-minute interval.

act$steps2<-act$steps
act$round_int_steps <- round(act$interval_steps, digits=0)
for(i in 1:length(act$steps)) {
    if(is.na(act$steps2[i])) { 
        act$steps2[i]<-act$round_int_steps[i]
    }
}
sum(is.na(act$steps2))

## [1] 0
